Summary: Latitude vs. X

The purpose of this project was to analyze trends in weather data based on latitude. Data was gathered using the OpenWeatherMap API to assemble a dataset of over 500 random locations across the globe.

After assembling the dataset, a Pandas dataframe was created and a series of graphs were drafted using Matplotlib that display Max Temperature, Humidity, Cloudiness, or Windspeed on the Y-Axis and Longitude on the X-Axis. The source data for the graphs is provided in this site along with detailed explanations of each graph.
